# Library Management System (MERN)

A full-stack role-based library management system built using MongoDB, Express, React, and Node.js.

## Features
- Multi-admin authentication
- Book issue/return
- Fine payment
- Reports dashboard
- Protected routes
- Responsive UI

## Tech Stack
- React
- Node.js
- Express
- MongoDB
- JWT Authentication

## Setup

Backend:
cd backend
npm install
npm run dev

Frontend:
cd frontend
npm install
npm run dev

Create .env file in backend:

PORT=5000
MONGO_URI=your_mongo_connection
JWT_SECRET=your_secret
